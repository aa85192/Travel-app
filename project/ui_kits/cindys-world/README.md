# Cindy's World — UI Kit

A hi-fi interactive recreation of Cindy's Paradise (Korea travel planner) shown in a 390×844 phone frame.

## Files
- `index.html` — interactive multi-screen prototype (Home → Itinerary → Map → Budget → Settings)
- `Components.jsx` — shared UI atoms (Pill, Card, BearIcon, FlightCard, etc.)
- `screens/Home.jsx`, `screens/Itinerary.jsx`, `screens/Map.jsx`, `screens/Budget.jsx`, `screens/Settings.jsx`
- `App.jsx` — root with bottom-nav state machine

Open `index.html` to use the prototype.

## Source mapping
- Pulled visuals & data shape from `aa85192/Travel-app` `src/index.css`, `src/App.tsx`, `src/components/layout/BottomNav.tsx`, `src/pages/Home.tsx`, `src/data.ts`.
- Bear PNG icons copied from `assets/bear-icons/`.
- Sticker illustrations (empty states + celebrations) from `assets/stickers/`.
