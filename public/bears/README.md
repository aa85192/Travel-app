# 熊熊圖檔

請把客製化的熊熊圖（已去背的 PNG 或 WebP）放在這個資料夾。

## 命名規則

| 用途 | 檔名 | 對應頁籤 |
|---|---|---|
| 待辦清單熊（行程頁籤） | `itinerary.png` | 行程 |

## 規格

- **格式**：PNG 或 WebP（去背）
- **尺寸**：建議正方形，至少 256×256（會自動縮）
- **背景**：透明（去背）

## 加更多

如果之後想換其他熊（首頁、地圖、預算、待辦、設定），命名照樣：
`home.png` / `map.png` / `budget.png` / `todo.png` / `settings.png`

不存在的檔案 App 會自動 fallback 到內建 SVG 熊。
